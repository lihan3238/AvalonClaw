# AI Avalon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first local playable AI Avalon app for one human plus AI players.

**Architecture:** A Vite React app owns UI and local game state. Pure TypeScript modules own rules and AI prompt/validation. A Vite dev-server API route calls an OpenAI-compatible `/chat/completions` endpoint using `.env` secrets.

**Tech Stack:** React, TypeScript, Vite, Vitest, Testing Library, Zod, Lucide React, Node fetch.
